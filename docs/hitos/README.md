contiene la documentación original del hito (descripción, decisiones de diseño, diagramas y casos).

| 8 | Patrones estructurales — Adapter, Bridge, Composite, Decorator, Facade, Flyweight, Proxy | [hito08-estructurales](hito08-estructurales/) |
| 9 | Diagramas UML consolidados | [hito09-diagramas](hito09-diagramas/) |
